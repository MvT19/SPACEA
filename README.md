# SPACEA
SPACEA is a custom-made GIS toolbox based on python scripts for use in ArcGIS. SPACEA is short for finding "<b>s</b>uitable s<b>pac</b>e in the s<b>ea</b>" and can be used to perform basic GIS analyses for <a href="https://marinebiodiversitymatrix.org/wiki/Marine_spatial_planning_definition">marine spatial planning</a>.
